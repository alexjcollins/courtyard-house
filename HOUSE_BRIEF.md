# Courtyard House Brief

This file is the current written reference brief for the house layout, room areas, and key glazing assumptions. Future agent runs should use this file as the primary descriptive reference for the project unless the user explicitly replaces part of it.

## Status

- This is a working design brief for the current scheme.
- Where this file conflicts with older rough assumptions elsewhere in the repo, this file should take precedence for room areas, garage size, and the opening schedule described below.

## Core Form

- The house is arranged around a central open courtyard.
- The courtyard is external, open to the elements, and has a soil base.
- The courtyard is not internal accommodation and should not be counted as part of the internal building area.
- The internal arrangement includes a connected garage, entrance sequence, a large open kitchen / dining / living volume, an office, a laundry room, circulation space, a principal bedroom suite, and two guest bedrooms.

## Area Summary

### Internal area excluding garage

- Total internal house area excluding garage: `205.0 sqm`

### External courtyard

- Central courtyard: `25.0 sqm`

## Design Interpretation Notes

- The kitchen / dining / living room is the principal shared family space and should be treated as a large single open-plan volume.
- The office is a dedicated separate room, not an informal corner of the main living space.
- The powder room is a compact downstairs WC.
- The garage is part of the connected built form, but it remains secondary accommodation.
- The courtyard is central to the scheme spatially and visually, but it remains external and unconditioned.
